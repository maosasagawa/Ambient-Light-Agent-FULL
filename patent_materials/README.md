# Patent Materials Package

This directory contains a professional first-pass patent package for the natural-language-to-matrix ambient lighting technology in this repository.

Files:

- `01_technical_disclosure.md`: Technical disclosure document suitable for internal review or transfer to a patent agent.
- `02_patent_draft_cn.md`: Chinese patent draft containing title, abstract, claims, specification, and embodiment structure.
- `03_figures_and_experiments.md`: Suggested figure set, comparison experiments, and evidence collection plan.
- `04_inventor_questionnaire.md`: Inventor fill-in sheet for prosecution preparation and novelty strengthening.

Current positioning:

- Recommended filing route: invention patent.
- Recommended claim focus: natural-language-driven generation of pre-diffusion LED matrix effects for a panel with an acrylic or other diffusion plate.
- Recommended novelty emphasis: semantic planning + low-resolution matrix constraints + optical diffusion pre-compensation + hardware-direct RGB output.

Important:

- These materials are written to be practically usable by a patent agent, but they are not a legal opinion.
- Before formal filing, add product photos, calibration data, comparison experiments, and inventor confirmation on the true earliest implementation date.
